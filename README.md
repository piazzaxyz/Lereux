## ◈ &nbsp;Sobre o Projeto

**LEREUX** é um portfólio digital criado para apresentar o trabalho e a direção criativa de **Lereux** — profissional atuante na cena trap como **Social Media Strategist** e **Stylist**.

A plataforma centraliza colaborações criativas, estratégias de branding, conceitos visuais e projetos de styling conectados à **estética urbana contemporânea** — com referências diretas à cultura trap francesa, design editorial minimal e o universo Y2K digital.

## ◈ &nbsp;Instalação

```bash
# Clone o repositório
git clone https://github.com/yourusername/lereux.git

# Entre no diretório
cd lereux

# Instale as dependências
pnpm install

# Inicie o servidor de desenvolvimento
pnpm dev
```

Acesse **`http://localhost:3000`** no navegador.
